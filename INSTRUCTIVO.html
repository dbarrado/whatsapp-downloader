Instalación: descarga automática de archivos desde WhatsApp

Este programa conecta tu WhatsApp al sistema de conciliación. Cada imagen o PDF que llegue al grupo designado se descarga automáticamente en tu computadora.

Importante: usá un número de WhatsApp de prueba para esta instalación, no el número principal del estudio. Se recomienda un chip secundario o prepago exclusivamente para este fin.
Paso a paso
1
Instalar Node.js (si no lo tenés)
Node.js es el motor que corre el programa. Si ya lo instalaste en alguna sesión anterior del curso, saltá al paso 2.

Descargá la versión LTS desde: nodejs.org → hacé clic en el botón verde grande que dice "Download Node.js (LTS)" → instalalo con todas las opciones por defecto (siguiente, siguiente, instalar).
✅
Para verificar que quedó instalado: abrí una ventana de comandos (tecla Windows → escribí cmd → Enter) y escribí node --version. Tiene que aparecer algo como v20.x.x.
2
Descargar y descomprimir el proyecto
Diego te comparte una carpeta llamada whatsapp-geodelta.zip por Google Drive.
Descargala y descomprimila (clic derecho → Extraer todo) en un lugar que recuerdes, por ejemplo en el Escritorio o en Documentos.
📁
Dentro vas a ver tres archivos: index.js, package.json y este instructivo. Eso es todo lo que necesitás.
3
Abrir la terminal dentro de la carpeta
Entrá a la carpeta que descomprimiste. Con la carpeta abierta en el Explorador de Windows, hacé clic en la barra de direcciones (arriba, donde dice la ruta) → escribí cmd → Enter.

Se abre una ventana negra de comandos ya posicionada en la carpeta correcta.
💡
Alternativa: tecla Windows → escribí cmd → Enter → y luego escribí cd seguido de la ruta completa a la carpeta (podés arrastrar la carpeta a la ventana negra para pegar la ruta).
4
Instalar las dependencias
En la ventana negra, escribí el siguiente comando y presioná Enter:
npm install
Esto descarga automáticamente todo lo que el programa necesita, incluyendo un navegador Chrome interno (~300 MB). Puede tardar entre 3 y 8 minutos dependiendo de la velocidad de internet. Esperá hasta que vuelva a aparecer el cursor parpadeante.
⏳
Si ves mensajes que pasan rápido con texto en verde o amarillo, es normal. Solo hay problema si aparece un mensaje en rojo que dice ERROR. En ese caso, avisale a Diego.
5
Correr el programa
Una vez que terminó el paso 4, escribí:
npm start
El programa arranca. La primera vez tarda unos 20-30 segundos en aparecer el código QR. Vas a ver algo así en la pantalla:

6
Escanear el QR con tu celular
Abrí WhatsApp en el celular de prueba y seguí este camino:
⋮ Menú → Dispositivos vinculados → Vincular dispositivo
Apuntá la cámara al código QR que aparece en la pantalla negra. En unos segundos vas a ver en la terminal:
✅ Autenticado. La sesión queda guardada — el próximo inicio no necesita QR.
🟢 Conectado a WhatsApp Web. Escuchando archivos de todos los grupos.
🔁
La sesión queda guardada. La próxima vez que corras npm start no necesitás escanear el QR de nuevo.
Configuración opcional — filtrar por grupo

Por defecto el programa escucha todos los grupos. Si querés que solo descargue de un grupo específico, abrí el archivo index.js con el Bloc de notas y buscá esta línea al principio:

const GRUPO = ''; // vacío = todos los grupos

Reemplazá las comillas vacías con el nombre exacto del grupo (copialo de WhatsApp para no equivocarte):

const GRUPO = 'Cobranzas Geo Delta';

Guardá el archivo y volvé a correr npm start.

¿Dónde quedan los archivos descargados?

Dentro de la carpeta del programa, se crea automáticamente una carpeta llamada downloads/. Cada archivo se guarda con el nombre en formato:
2026-06-22_10-30-00_NombreContacto.jpg

La terminal también muestra un mensaje cada vez que llega y se descarga un archivo nuevo.

Si algo no funciona
❌
El QR no aparece o tarda demasiado: cerrá la terminal con Ctrl+C y volvé a correr npm start.
❌
Error en npm install: capturá una foto de la pantalla y mandásela a Diego antes de la sesión del lunes.
❌
El QR apareció pero no pude escanearlo a tiempo: el QR expira en ~30 segundos. Cerrá con Ctrl+C y volvé a correr npm start para generar uno nuevo.
Opción avanzada — funcionar aunque la PC esté apagada

La instalación que hiciste en los pasos anteriores funciona solo mientras tu computadora está encendida y el programa está corriendo. Si cerrás la ventana o apagás la PC, los archivos dejan de descargarse.

Si necesitás que el sistema funcione las 24 horas sin depender de ninguna computadora, la solución es instalarlo en un servidor en la nube. Acá explicamos dos opciones para hacerlo.

¿Qué es un servidor en la nube?

Imaginá una computadora que nunca se apaga, está conectada a internet las 24 horas y la alquila una empresa por un precio mensual fijo. Eso es un servidor en la nube. Vos instalás tu programa ahí, y corre solo — sin que nadie tenga que hacer nada.

OPCIÓN A
Railway — el camino más fácil
~USD 5–10 / mes

Railway es una plataforma que instalas el programa con un clic y ellos se encargan de todo: el servidor, los backups, que no se caiga. No hace falta saber nada técnico para usarlo.

Cómo funciona:
1. Creás una cuenta en railway.app
2. Hacés clic en "Deploy" sobre la plantilla de Evolution API
3. Conectás tu WhatsApp escaneando un QR (igual que antes)
4. Listo — el sistema corre solo, las 24 horas
💡
Diferencia con la instalación local: en vez de correr en tu PC, corre en un servidor de Railway. Vos solo ves los resultados (archivos descargados) desde cualquier lugar.
OPCIÓN B
VPS propio — más control, mismo precio
~USD 4–12 / mes

Un VPS (servidor privado virtual) es como alquilar tu propia computadora en la nube. Tenés más control que con Railway, pero también hay que configurar más cosas al principio. Proveedores populares: Hetzner, DigitalOcean, Hostinger.

⚠️
Esta opción requiere ayuda técnica para la configuración inicial. Ideal si ya tienen alguien de sistemas en el equipo, o si Diego lo configura en una sesión aparte.
